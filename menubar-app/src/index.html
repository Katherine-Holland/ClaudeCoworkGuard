One-time setup.

This takes about 30 seconds and never needs to be done again.

01
Generate security certificate
Creates the CoworkGuard Security Certificate that allows traffic scanning.
02
Trust the certificate
Tells your Mac the CoworkGuard certificate is safe. You'll be asked for your password once.
03
Verify setup
Confirms everything is working correctly.
✅
CoworkGuard is ready
Use the shield icon in your menubar to start and stop protection.
