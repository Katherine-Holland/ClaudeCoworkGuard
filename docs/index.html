Features

Everything Cowork
should have built in.

The audit trail, scanner, and domain guard that Anthropic haven't shipped yet.

🔍
Payload Scanner
Every outbound request to api.anthropic.com is scanned against 36 patterns before leaving your machine. SSNs, API keys, private keys, DB credentials — caught and blocked.
⚠️
Domain Guard
Detects when Cowork is active and fires an in-page warning banner the moment you navigate to AWS Console, Gmail, Stripe, GitHub, Salesforce, or 15+ other sensitive domains.
📋
Live Audit Log
Every intercepted request is logged locally — timestamped, categorised, and filterable. The compliance trail Cowork doesn't provide. Export to JSON any time.
📊
Payload Trend Chart
24-hour bar chart of data volume sent to Claude, colour-coded by risk level. Spot unusual spikes at a glance. See exactly when your sensitive data exposure is highest.
⚙️
Settings Panel
Toggle block levels for Critical, High, and Medium severity independently. Add custom regex patterns for your organisation's internal formats. No config file editing required.
🔒
Zero Cloud Dependency
Everything runs on your machine. No accounts. No telemetry. No analytics. Raw payload content is never stored — only SHA-256 hashes and redacted previews.

How it works

Set up in 4 steps.

A local proxy + Chrome extension. Everything stays on your machine.

1
Start Proxy
Run mitmproxy on localhost:8080. Trust the certificate in Keychain.
2
Set System Proxy
Point macOS HTTP/HTTPS proxy to 127.0.0.1:8080 in Network settings.
3
Load Extension
Install from Chrome Web Store or load unpacked from the GitHub repo.
4
Open Dashboard
Visit localhost:7070. Your audit trail starts the moment Cowork does.
